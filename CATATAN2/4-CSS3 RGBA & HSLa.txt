RGBa 
    Red Green Blue Alpha

    Alpha (tranparansi)

RGBa digunakan untuk membuat tranparansi warna, nilai dari 0 - 255,
untuk membuat warna hitam rgb(0,0,0);
untuk membuat warna putih rgb(255,255,255);

misal 
    warna orange -> kode hexanya #ff6600, diganti rgba -> rgba(255, 102, 0, 0.5)
    r -> Red
    g -> Green
    b -> Blue
    a -> Alpha (transparansi warna)


     .satu {
        background-color: #ff6600;
    }

    -> ditambah transparansi warna
    .satu {
    background-color: rgba(255, 102, 0, 0.5);
    }


HSLa -> Hue Saturation Lightness Alpha 

    -> Nilai Hue, nilai nya 0 - 360, 0 untuk merah, 120 untuk hijau, dan 240 untuk biru. 

   -> Nilai Saturation 0 - 100%, semakin tinggi nialinya semakin murni warnanya, kalau turun warna asli agak memudar.0% berarti bayangan abu2, and 100% ful  color. 

   -> Nilai Lightness 0 - 100%, semakin tinggi warnanya semakin terang
